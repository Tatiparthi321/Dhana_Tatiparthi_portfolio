Lung Cancer Detection using LSTM and MobileNet
This project is a deep learning-based system designed to detect lung cancer from CT scan images. It uses a hybrid architecture combining MobileNet (a lightweight Convolutional Neural Network) for efficient image feature extraction and Long Short-Term Memory (LSTM) networks for learning temporal dependencies in image sequences.

The model is trained on a dataset categorized into benign, malignant, and normal cases. It aims to support medical professionals by providing a reliable and efficient tool for early diagnosis of lung cancer using computer vision and AI.

Research Paper Fetching Tool using PubMed API
This project involves building a Python-based command-line tool that fetches biomedical research papers from PubMed based on user-defined queries. It intelligently parses XML responses to extract metadata like title, publication date, affiliations, and corresponding author email. The results are filtered to identify non-academic authors linked to pharmaceutical or biotech companies and saved into a structured CSV format.
